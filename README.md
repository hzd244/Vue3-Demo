## 项目介绍

该项目采用 Vue3 技术栈实现并汇总了一些案例 Demo，并且部分案例在掘金上同步更新具体的实现细节

使用 unplugin-vue-router 根据 views 目录自动生成路由配置（文件夹的名称为访问的路径），不需要手动编写路由配置文件，views 目录下每个文件夹代表一个案例

## 克隆项目

```shell
git clone https://github.com/hzd244/Vue3-Demo.git

```

## 安装依赖

```bash
cd vue3-store-tools

pnpm install

```

## 运行项目

```bash
pnpm dev
```
